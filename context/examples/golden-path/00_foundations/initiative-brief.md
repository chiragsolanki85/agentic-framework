# Initiative Brief

## Context summary

- North star reference: Reduce onboarding drop-off for SMB customers.
- Problem definition reference: 42% users abandon setup before first value.
- Persona and journey references: Owner-Ops persona and first-week activation journey.
- Target outcomes reference: Improve activation completion by 20% in 8 weeks.

## Initiative definition

- Initiative name: Guided onboarding and first-value acceleration
- Value hypothesis: Contextual guidance and progressive setup will increase first-week activation and reduce support burden.
- Scope boundaries: Web onboarding only; excludes billing redesign.
- Key risks: Over-guidance causing friction, analytics blind spots.
- Dependencies: Event tracking pipeline and feature-flag service.
