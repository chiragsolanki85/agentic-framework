## Feature ID policy

Each feature uses one stable ID:

`FEAT-<EPIC>-<NNN>`

## Mandatory links

- Requirement file
- Spec file
- Implementation log
- PR description references

## Why it matters

Traceability enables:

- deterministic audits
- faster root-cause analysis
- confident approvals under enterprise controls
