## Governance rules

1. Artefacts are source of truth.
2. No coding starts without approved context.
3. Every stage transition is logged.
4. Feature closure requires context completeness.

## What this prevents

- hidden implementation outside approved intent
- skipped gates during deadline pressure
- incomplete closure where docs diverge from code
