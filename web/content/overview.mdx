## Purpose

This portal presents the Crosstide Agentic SDLC blueprint as an enterprise-ready operating guide.

## Intended audience

- Product Managers defining outcomes and constraints
- Engineers and Architects executing technical delivery
- Delivery and Governance Leads approving risk, controls, and release readiness

## How to use this portal

1. Complete foundations across all domains.
2. Run features through the full lifecycle.
3. Enforce mandatory human gates.
4. Use Core first, then adopt Extended controls as needed.
