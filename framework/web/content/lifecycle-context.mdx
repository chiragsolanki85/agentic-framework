## Purpose

Ensure repository context stays aligned to deployed reality. A feature is complete only when code **and** context are both complete. Performs explicit diff and consistency checks before closure.

[Explore this stage in the SDLC Flow](/flow#context-lifecycle)

## Inputs

Merged and deployed feature artefacts, current architecture/contracts, and feedback synthesis.

## Outputs

Context diff, consistency check, and closure log entry.

## Prompt to run

`framework/prompts/14-context-lifecycle-agent.md`

## Human gate

**Owner:** Engineering + Product  
**Decision:** Verify context complete and close the feature loop.
