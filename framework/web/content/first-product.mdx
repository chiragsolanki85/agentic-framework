## Project kickoff

- Complete all `context/00_foundations/**/*.md` files.
- Confirm the checklist in `context/00_foundations/cross-cutting/foundations-checklist.md`.
- Record approval in `context/logs/handoffs/30_handoffs.md`.

## Required gates

- Refined-ready
- Spec approval
- PR approval
- Validation approval
- Deploy go/no-go
- Outcome triage
- Context complete verification
