## Core categories

- Performance SLOs
- Reliability/SLA targets
- Security controls and review triggers
- Data retention and privacy constraints

## Release expectation

Validation and deployment approvals should explicitly confirm:

- no critical security exceptions
- NFR thresholds are met
- privacy and retention constraints are respected
